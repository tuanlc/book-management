- Database: PostgreSQL
- ORM: Gorm
- Language: Go
- Hot reload: Air

## Development
### Prequisite
- Install hot reload tool: https://github.com/cosmtrek/air

```sh
make deps
```

### Run
```sh
$ make run
```
